How the Model Works
📊
Data Collection
Historical commodity prices from 2021–2025 are stored in SQLite database. USD prices are converted live to INR.
🧮
Linear Regression
The model fits a straight trend line through all historical data using Ordinary Least Squares (OLS) — no ML libraries needed.
🔮
Forecasting
The fitted line is extended forward in quarterly steps to estimate future prices. Results should be used as indicative only.
Disclaimer: Predictions are based on simple linear regression and do not account for war escalations, policy changes, or sudden shocks. For educational use only.
